## ✨ Features

- 🔐 **JWT Authentication** — Secure signup & login with hashed passwords and persistent sessions
- 📊 **Visual Charts** — Spending breakdown by category via Recharts (bar, pie, trend)
- 💳 **Transaction Management** — Add, view, and delete income/expense entries with categories
- 📈 **Dashboard Summary** — Real-time balance, net income, and total expenses at a glance
- 🔒 **Protected Routes** — All pages gated behind auth; unauthenticated users auto-redirected
- 🗂️ **Category Filtering** — Filter transactions instantly by food, rent, salary, and more

## 🚀 Quick Start

```bash
# 1. Clone the repo
git clone https://github.com/your-username/expense-tracker.git

# 2. Install dependencies
cd client && npm install
cd ../server && npm install

# 3. Set up environment variables in /server/.env
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key

# 4. Start both servers
npm run dev   # backend on :5000
npm run dev   # frontend on :5173
```
